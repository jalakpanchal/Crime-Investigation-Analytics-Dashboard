# 🔍 Crime Investigation Analysis and Visualization

> An analytical Power BI project that uncovers crime patterns, trends, and hotspots through interactive dashboards and data-driven visual storytelling.

---

## 📌 Overview

This project performs end-to-end analytical visualization of crime-related datasets to help identify meaningful patterns across categories, areas, and time periods. The goal is to transform raw crime data into clear, decision-supportive insights using Power BI dashboards.

---

## 🎯 Objectives

- Identify high-frequency crime categories and area-wise distribution
- Detect temporal patterns (time of day, day of week, seasonal trends)
- Present findings in an intuitive, interactive dashboard format
- Support data-driven decision-making for investigation and prevention

---

## 🛠️ Tech Stack

| Tool | Purpose |
|------|---------|
| **Power BI** | Dashboard creation and visualization |
| **Microsoft Excel** | Data cleaning and preprocessing |
| **DAX** | Calculated measures and KPIs |

---

## 📊 Dashboard Features

- **Category-wise Analysis** — Breakdown of crime types by frequency and severity
- **Area-wise Heatmaps** — Geographic distribution of incidents across zones
- **Trend Analysis** — Monthly and yearly crime trend charts
- **KPI Cards** — Total incidents, top crime category, most affected area
- **Interactive Filters** — Slice data by year, area, crime type, and time

## 🚀 How to Run

1. Clone this repository
   ```bash
   git clone https://github.com/jalakpanchal/crime-investigation-analysis.git
   ```

2. Open `dashboards/crime_analysis.pbix` in **Power BI Desktop**

3. If prompted, update the data source path to point to `data/crime_dataset.csv`

4. Click **Refresh** to load the data and explore the dashboards
